## Description
A weather application built with React that provides real-time weather information for different cities. The app fetches data from a weather API and displays temperature, humidity, and weather conditions.

## Features
- Search weather by city name
- Real-time weather data
- Displays temperature and humidity
- Dynamic interface using React state

## Technologies Used
- React
- JavaScript
- HTML
- CSS
- Weather API
